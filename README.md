# Media Oiljang

Media Oiljang is an experimental archive and community-based media commons project.

This repository documents:
- local storytelling practices
- participatory filmmaking
- AI media experiments
- cultural democracy research
- temporary creative communities
- locality-based cinema movements

Rather than focusing only on views, profit, or platform growth,  
this project explores slower forms of media practice rooted in relationships, place, and shared experience.

Based in Busan, Korea.

## Ongoing Topics

- local cinema
- AI and community media
- cultural commons
- amateur performance and everyday acting
- moving gatherings and temporary communities
- storytelling as social practice
